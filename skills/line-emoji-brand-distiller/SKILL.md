# LINE絵文字 brand distiller

## Workflow

1. `../../rules/structure-constraints.md` で構造成立性を確認する。
2. `../../rules/brand-taxonomy.md` と `../../rules/brand-creation-rules.md` を確認する。
3. `../../workflows/brand-distillation-workflow.md` を開き、抽出手順を固定する。
4. `../../templates/brand/brand-setting-template.md` と `../../templates/brand/brand-positioning-template.md` を使う。
5. 制作へ進むなら `../../templates/brand/brand-production-brief-template.md` へ落とす。

## Output Rules

- `構造判定 / ブランド判定 / 商品判定 / positioning / Design Ready条件 / 次アクション` を出す。
- ブランド設定だけ濃く、商品へ落ちない案は `Revise` にする。
