# LINE絵文字 market scout

## Workflow

1. `../../PROJECT_MAP.md` を開き、market の正本範囲を確認する。
2. `../../rules/market-observation-rules.md` を開き、観測証跡の必須項目を確認する。
3. `../../workflows/market-observation-workflow.md` を開き、観測手順を固定する。
4. `../../templates/market/market-observation-log-template.md` と `../../templates/market/category-gap-map-template.md` を使う。
5. 観測できた差別化軸を `../line-emoji-brand-distiller/SKILL.md` へ渡す。

## Output Rules

- `観測範囲 / 混雑領域 / 空白領域 / 模倣禁止境界 / 差別化軸 / evidence strength` を出す。
- 最新市場を語る場合は観測日を明記する。
- 単発観測を恒久 rule にしない。
