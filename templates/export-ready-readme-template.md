# Export Ready

このディレクトリは、ユーザーが手作業で LINE Creators Market へ登録する直前の提出候補 asset を置く場所である。

## 原則
- final candidate と export-ready asset を混ぜない。
- ファイル名は選択した package type の番号に合わせる。
- トークルームタブ画像とコンテンツ画像を混同しない。
- `submissions/sales-package-manifest.md` と必ず対応させる。

## 置くもの
- 公式仕様に合わせた提出候補画像。
- 必要なら ZIP 化した提出候補。
- 再提出時は前回版を上書きせず、manifest で版を分ける。
