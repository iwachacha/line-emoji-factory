# Submission Checklist

## Assets
- [ ] Content images are PNG/APNG.
- [ ] Content images match the selected item type size.
- [ ] Static emoji content images are 180 x 180 px.
- [ ] Static sticker content images are no larger than 370 x 320 px and use even-numbered dimensions.
- [ ] Static sticker main image is 240 x 240 px when included.
- [ ] Tab image is 96 x 74 px when included.
- [ ] Background is transparent.
- [ ] File size is within LINE limits.
- [ ] Content image count matches package type.

## Metadata
- [ ] Creator name is within limit.
- [ ] Title is within limit.
- [ ] Description is within limit.
- [ ] Copyright is within limit and valid.
- [ ] Metadata does not contain emoji.
- [ ] Metadata matches the release contents.

## Review Risk
- [ ] No forbidden advertising expression.
- [ ] No unauthorized third-party IP.
- [ ] No specific-person-only expression.
- [ ] No simple company logo package.

## Records
- [ ] Validation report is attached.
- [ ] Submission audit report is updated.
- [ ] Release log is updated.
- [ ] Watch items are moved to quality ledger.
