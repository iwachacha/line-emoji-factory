## 4. Repeating Failure Patterns
- 近い絵文字同士で混ざる箇所: `了 / 済 / 決`, `中 / 調`, `見 / 確 / 返`
- 句読点や既存記号で十分になってしまう箇所: `急`, `締` は `!` や `※` に負けやすいが、低圧さで残す理由がある
- スタンプや他商品に負ける箇所: 強い感謝やおつかれ表現はスタンプに負ける。`礼`, `休` は補助枠として扱う
- 仕事寄り / 日常寄りの偏り: 仕事寄りが強い。説明文で「日常」を入れ、予定調整シーンも検証する

## 5. Decision Summary
- 残す絵文字: `001` to `016`
- 強く直す絵文字: `002`, `004`, `013`, `015` は混同監査を重点化
