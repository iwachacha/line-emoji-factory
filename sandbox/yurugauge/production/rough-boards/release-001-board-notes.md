# release-001 rough board notes

## Rough Goal
- 4 x 4 の16個で、返答・進行・保留・締めの状態差分が即読できるかを見る。
- 「文字敬語」ではなく「状態ゲージ」として買う理由が立つかを見る。

## Visual Anchor
- 太い角丸ゲージ
- 中央の1文字ラベル
- 下部の進捗バー
- 1個だけの補助サイン
- 低圧のくすみ色ではなく、視認性を保つ明るいアクセント

## Keep
- 外形ゲージと下部バーは全絵文字で固定する。
- ラベルは大きく、1文字を原則にする。
- 透明余白を少なくし、公式推奨どおり画像を大きく使う。

## Trim
- 細いサイン、複数アイコン、長い文字、装飾影は削る。
- `ありがとう` や `休憩` の装飾はセット内で浮かない強さに抑える。

## Drift Watch
- `了 / 済 / 決`: 返答、処理完了、確定事項の差を保つ。
- `中 / 調`: 進行中と調整中をバー量と補助サインで分ける。
- `見 / 確 / 返`: 確認済み、確認依頼、返信依頼の向きを分ける。
