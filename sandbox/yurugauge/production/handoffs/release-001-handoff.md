# release-001 制作ハンドオフ

## 1. 基本情報
- ブランド名: `ゆるゲージ`
- release / set 名: `進捗と返事コア16`
- 参照するブランド設定: `brand/brand-setting.md`
- 参照する制作基盤: `brand/brand-production-brief.md`
- 参照する set 仕様: `emoji-sets/releases/release-001.md`
- 参照する quality ledger: `qa/quality-ledger.md`
- 参照する usage validation: `qa/usage-validations/release-001.md`
- 判定状態: `Design Ready`

## 2. rough stage の要約
- rough stage の目的: 16個の状態が同じゲージ文法で揃い、混同しやすい語が区別できるか確認する。
- 使用した rough board / 全体図: `production/rough-boards/release-001-board-notes.md`, `production/finals/release-001-preview.png`
- 固定できた視覚記号: 角丸ゲージ、1文字ラベル、下部バー、補助サイン、太線。
- 固定できた差分軸: ラベル、バー量、色、補助サイン。
- まだ未確定の点: 公開後にどの検索語が効くか。`休` が初回セットに残るか。
- ClaudeDesign に渡す理由: コード生成のベクター風完成候補を基準にしつつ、小表示と質感の最終監査を人間側でできるようにするため。

## 3. set 全体で守ること
- 単体送信時の成立条件: 1個で「状態返答」として読める。
- 文中使用時の成立条件: 短文の前後に置いても、文字本文の邪魔をしない。
- 連続使用時の成立条件: 2個までなら状態推移として読める。
- 絶対に残す要素: 横長ゲージ外形、太い輪郭、中央ラベル、下部バー。
- 削ってよい要素: 細かすぎる補助サイン、薄い影、色の微差。
- 禁止 drift: キャラクター化、長文ラベル化、ハンコ風への後退、企業UI風。
- 余白 / アウトライン / 装飾の補足: 外形は大きく、輪郭は濃く、補助サインは1つだけ。

## 4. ClaudeDesign への共通指示
- 仕上げの目的: LINE絵文字として小表示で読める、販売用の個別PNGへ仕上げる。
- 1絵文字ごとに優先する順序: `意味の即読性 → 小表示可読性 → セット整合性 → 装飾`
- rough からの変更許容範囲: 色調、字間、バー幅、補助サインの簡略化は可。
- 変更してはいけない核: ラベルと状態意味、ゲージ外形、下部バー。
- 小表示で最優先する点: 中央ラベルが潰れないこと。
- set 全体で揃える点: 輪郭太さ、バー位置、角丸、透明余白。
- 未解決 `Watch` 項目: `了 / 済 / 決`、`中 / 調`、`見 / 確 / 返` の混同。
- 生成後に優先監査する絵文字: `002`, `003`, `004`, `005`, `006`, `013`, `015`
- タブ画像と metadata への注意: タブ画像は `進` のゲージでブランド核を示す。metadata は告知文言と絵文字を含めない。

## 5. 個別絵文字仕様
- `001.png / 見`: 見ました。バー35%。小さな目サイン。`確` と違い、依頼ではなく確認済み。
- `002.png / 了`: 了解。バー70%。チェックサイン。`済` より返答寄り。
- `003.png / 中`: 対応中。バー55%。進行サイン。`調` より作業進行。
- `004.png / 済`: 完了。バー100%。強いチェック。`決` より処理完了。
- `005.png / 確`: 確認して。バー45%。虫眼鏡サイン。依頼方向を出す。
- `006.png / 返`: 返信ください。バー40%。戻り矢印。圧を強くしすぎない。
- `007.png / 共`: 共有です。バー60%。接続点サイン。周知の意味。
- `008.png / 礼`: ありがとう。バー80%。小さな光サイン。装飾は控えめ。
- `009.png / 待`: 保留。バー25%。一時停止サイン。止める意味。
- `010.png / 後`: あとで。バー30%。時計サイン。放置ではない。
- `011.png / 急`: 急ぎ。バー90%。感嘆サイン。赤を使うが警告標識に寄せない。
- `012.png / 締`: 締切。バー85%。カレンダーサイン。`急` より期限。
- `013.png / 調`: 調整中。バー50%。スライダーサイン。`中` と差を出す。
- `014.png / 未`: 未定。バー15%。疑問サイン。ネガティブに寄せすぎない。
- `015.png / 決`: 決定。バー100%。丸印サイン。`済` より確定事項。
- `016.png / 休`: 休憩。バー10%。休止サイン。初回セットの温度調整枠。

- rough / final の保存先: `production/rough-boards/`, `production/finals/release-001/`
- 修正依頼の記録先: `qa/quality-ledger.md`

## 7. 差し戻し条件
- Stage 2 で直す問題: 文字のにじみ、輪郭太さ、バー幅、補助サインの見え方。
- Stage 1 へ戻す問題: ラベルや状態カテゴリの混同、セット全体の文法崩れ。
- 構造再評価へ戻す問題: 単体で意味が読めない絵文字が多数出る、配置依存でしか成立しない。
