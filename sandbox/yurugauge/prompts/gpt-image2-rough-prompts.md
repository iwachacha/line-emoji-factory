# ゆるゲージ GPT-image2 rough prompts

## Core Rough Prompt
LINE emoji rough board for a Japanese static emoji brand named "ゆるゲージ". Create a 4x4 board of simple rounded progress-gauge symbols. Each emoji is a transparent-background 180x180 style icon with thick dark outline, a central large Japanese single-character label, and a bottom progress bar. Tone is practical, soft, low-pressure, not a stamp, not a UI logo. Labels: 見, 了, 中, 済, 確, 返, 共, 礼, 待, 後, 急, 締, 調, 未, 決, 休.

## Variant Rough Prompts
- Check that `了`, `済`, and `決` do not look like the same meaning. `了` is acknowledgement, `済` is completed task, `決` is decided.
- Check that `中` and `調` are different. `中` is in progress. `調` is being coordinated.
- Check that `見`, `確`, and `返` express different directions: already seen, please check, please reply.

## Negative / Guardrails
- Do not use long Japanese phrases inside the icons.
- Do not create a character face as the main subject.
- Do not use logos, app UI, notification badges, or brand-like marks.
- Do not rely on color difference only.
- Do not add sparkles, hearts, or decorative clutter except one tiny status sign when needed.
