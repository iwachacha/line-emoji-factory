# ゆるゲージ ブランド設定

この文書は、`ゆるゲージ` のブランド核を固定する正本である。
市場で強い `敬語 / 仕事 / 連絡` 需要に接続しながら、既存のテキスト敬語やはんこ風の混雑を避け、
「いまどの状態か」を一拍で伝えるゲージ記号へ絞る。

## 判定サマリー
- 判定状態: `Design Ready`
- 構造分類: `単体反応型`
- ブランド主型: `記号`
- ブランド副型: `図形・抽象`
- 補助タグ: `実用品・機能`
- 採用枠: `標準枠`

## 共通コア
- ブランド名案: `ゆるゲージ`
- 一言定義: 返信と進捗を、やわらかいゲージ記号で短く返せる状態共有絵文字。
- 想定ユーザー: LINEで予定調整、仕事連絡、家族連絡、グループ進行をする人。長文にせず「見た / 中 / 済 / 後」を返したい人。
- 主な使用場面: 確認済み、対応中、完了、保留、あとで対応、急ぎ、決定、調整、共有、感謝、締切。
- 得意な感情または機能領域: 進捗、返答、状態共有、軽い圧の調整。
- 不得意領域: 強い感情芝居、長文メッセージ、派手な祝祭、キャラクター人格、細かい説明。
- 視覚記号: 角丸ゲージ、太い濃色アウトライン、中央の短い漢字、下部の進捗バー、右上または左上の小さな状態サイン。
- 小サイズで守る要素: 外形ゲージ、1文字ラベル、太線、進捗バーの長さ、アクセント色。
- 絶対条件: 各絵文字が単体で意味を持つこと。文字は1文字または2文字以内。色違いだけで差分を作らないこと。ゲージの進み具合と中央ラベルの意味を一致させること。
- 可変条件: アクセント色、ゲージ量、状態サイン、角丸の深さ、補助アイコンの位置。
- 差分の軸: 返答系、進行系、保留系、締切系、感謝系。進捗量、中央ラベル、補助サインを組み合わせる。
- 変えてよいもの: 色、ゲージ量、補助サイン、ラベル字形の微調整。
- 変えてはいけないもの: ゲージ外形、太線、1文字即読、低温度、シンプルさ。
- NG表現: 細い線、長い文章、企業UIやロゴ風、アプリ通知風の誤認、過度なグラデーション、意味のない色違い量産。
- 商品化優先順位: 1. 見た / 了解 / 対応中 / 完了 2. 確認 / 要返信 / 共有 / 感謝 3. 保留 / あとで / 急ぎ / 締切 4. 調整 / 未定 / 決定 / 休憩。
- 出力トーン: 実用、やわらかい、軽い、整理された、低圧。
- 市場観測サマリー: 公式人気ではIP・アニメーション・テキスト絵文字が強く、クリエイターズ側では敬語・仕事・下線・ハート・シンプルキャラが多い。仕事敬語は混雑しているが、進み具合を統一ゲージで返す絵文字は相対的に薄い。
- 混みやすい表現: 敬語テキスト、はんこ風、四角ラベル、かわいいキャラ、アニメーション。
- 狙うホワイトスペース: 文字敬語そのものではなく、「今の状態」を短く見せる進捗ラベル。
- 尖り運用方針: 初期 release は通年の状態共有に限定する。感情温度や予定ピンは後続派生へ分ける。
- 派生候補: 感情温度版、予定調整版、家事タスク版、チーム連絡版、低文字量アイコン版。
- 量産リスク: 進捗バーが色違いに見える、漢字が小表示で読めない、仕事向けに寄りすぎて日常接続性が落ちる。
- 審査リスク: 画像内文字の誤字、タイトルや説明文との不一致、企業UIや外部サービス風の誤認。
- 商品化の勝ち筋仮説: 「了解です」「対応中です」と打つより軽く、スタンプより会話を止めない。短文の末尾や単体返信で再使用される。
- ブランド固有基盤化の要点: `1状態=1文字ラベル=1ゲージ量` を固定し、同じ状態の言い換えを増やさない。
- 次に詰めること: 完成PNGで `確認 / 要返信 / 了解`、`対応中 / 調整中`、`完了 / 決定` の混同を監査する。

## 記号で追加
- 形の文法: 外形は横長の角丸ゲージ。中央に1文字。下部に進捗バー。補助サインは小さく1つだけ。
- 線の性格: 太く濃い線。輪郭の黒をやや青緑に寄せ、硬すぎる事務UIを避ける。
- 反復ルール: 全絵文字でゲージ外形を反復し、状態差はラベル、進捗量、補助サインで出す。
- 連結時の見え方: 2個までなら状態の推移として読める。3個以上はノイズになるため推奨しない。

## 図形・抽象で追加
- 面と余白の比率: 絵柄を大きく使い、上下左右の透明余白は最小限。内側はラベル可読性のため余白を残す。
- 塊のリズム: 大きなゲージ面、太い輪郭、短いラベル、下部バーの四層。
- 禁止装飾: 粒ノイズ、過度な影、細かなアイコン複数、過剰な光彩。
- 反復単位: ゲージ外形と下部バーを固定し、状態サインだけを変える。

## 実用品・機能タグで追加
- 主要用途: 見た、了解、対応中、完了、確認、要返信、共有、感謝、保留、あとで、急ぎ、締切、調整、未定、決定、休憩。
- 高頻度シーン: 仕事連絡、PTAやグループ運営、家族の予定調整、友人との待ち合わせ、個人メモ。
- 代替されやすい既存表現: `OK`, `了解`, `見ました`, `対応中`, `済`, `あとで`, `!`, `※`, 既存の敬語絵文字。
- 実用優位点: 文章の温度を上げすぎず、状態だけを即読させられる。ゲージ量で「進んでいる / 止めている / 完了」を視覚的に補強できる。
