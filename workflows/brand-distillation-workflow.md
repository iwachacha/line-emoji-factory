# ブランド抽出ワークフロー

このファイルは、市場観測やアイデアからブランド核を作る標準手順を定義する。

## 起動条件
- アイデアをブランド設定へ落とす。
- 市場観測から差別化軸を抽出する。
- ブランド核が広すぎる、または商品へ落ちない。

## 使う正本
- ブランド分類: `rules/brand-taxonomy.md`
- ブランド創出判断: `rules/brand-creation-rules.md`
- 評価語彙: `rules/evaluation-model.md`
- 出力: `templates/brand/brand-setting-template.md`, `templates/brand/brand-positioning-template.md`, `templates/brand/brand-production-brief-template.md`

## 標準手順
1. ブランドを1文で言えるか確認する。
2. 主型 / 副型 / 補助タグを決める。
3. 視覚記号を3〜5個に絞る。
4. 使う場面と使わない場面を分ける。
5. 買う理由と release を増やせる余地を確認する。
6. 固定IPの場合は `workflows/ip-production-workflow.md` へ接続する。

## 完了条件
- `Design Ready` か `Revise` かを判断できる。
- brand setting と brand positioning に矛盾がない。
- production brief へ落とせる固定条件がある。
