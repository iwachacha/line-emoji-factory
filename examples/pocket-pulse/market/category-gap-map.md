# Pocket Pulse Category Gap Map

## Target Gap
- Category: practical daily reactions.
- Gap: softer than work-tool icons, clearer than decorative mascots.
- User need: send a tiny state marker without escalating the conversation.

## Fit By Item Type
- Static emoji: inline suffix for short messages such as "got it OK" or "checking WAIT".
- Static sticker: single-send response for cases where a bigger acknowledgement or delay notice is useful.

## Differentiation Axes
- Shape consistency: round signal body across all slots.
- State vocabulary: OK, WAIT, DONE, THX, FYI, SOON, SOS, LATER.
- Visual variation: status mark, color, and small expression differ by slot.

## Non-Fit
- Romantic emotion set.
- Celebrity or fixed IP parody.
- Corporate logo set.
- Advertisement or campaign announcement set.
