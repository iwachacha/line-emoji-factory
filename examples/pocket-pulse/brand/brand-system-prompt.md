# Pocket Pulse Production Prompt

You are producing assets for the LINE brand Pocket Pulse.

## Brand Core
- Shape: round signal mascot with a thick dark outline.
- Role: small daily status signals for lightweight coordination.
- Tone: calm, useful, warm, and not noisy.
- Must survive at emoji size and sticker size.

## Shared Visual Rules
- Keep the round core, two dot eyes, and one simple status mark.
- Use a limited but varied palette: teal, yellow, coral, blue, green, violet, red, and gray.
- Add a small shadow only inside the transparent canvas.
- Do not use real app names, corporate logos, release announcements, sale language, or personal targeting.
- Every asset must communicate one action or state in a single glance.

## Static Emoji Release
- Release: release-001
- Item type: static-emoji
- Canvas: 180 x 180 PNG, transparent background.
- Content: compact signal face plus short ASCII label.
- File source directory: releases/release-001/production/finals
- Companion tab: releases/release-001/production/tab/source-tab.png

## Static Sticker Release
- Release: release-002
- Item type: static-sticker
- Content canvas: up to 370 x 320 PNG, even width and height, transparent background.
- Main image: 240 x 240 PNG.
- Chat tab: 96 x 74 PNG.
- Content: larger signal face and clear short phrase for single-send use.

## Slots
1. OK - acknowledge.
2. WAIT - pause or hold.
3. DONE - completion.
4. THX - thanks.
5. FYI - light notice.
6. SOON - coming soon without promotional meaning.
7. SOS - help needed.
8. LATER - defer gently.

## Quality Gate
- Readability must hold on 180, 96, 48, and 32 px contact sheet previews.
- Each slot must differ by color and status mark.
- No slot may rely only on text.
- Transparent margin must remain balanced.
