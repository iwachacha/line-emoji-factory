# Pocket Pulse brand canon

## Core
- ブランドコアコンセプト: 小さな状態変化を、丸いパルス記号で即読できる実用ブランド。
- 守る核: 丸コア、アンテナ線、中央サイン、太い濃色アウトライン。
- 感情レンジ: 軽い確認、共有、完了、待機、ヘルプ。

## Guardrails
- 絶対に守る視覚要素: 丸い本体、中央サイン、太線、透明背景。
- 許容される変化: サイン、ポーズ、短い発話、表情の温度。
- 禁止 drift: 通信会社ロゴ化、UIアイコン化、細線、淡色、広告風、背景あり。

## Series Rules
- 継承すべき要素: 丸コアと太線。
- 新鮮さを出してよい軸: item type、発話の強さ、サインの用途。
- 避ける重複: OK / done / thanks などの高頻度反応を同じ温度で増やさない。
