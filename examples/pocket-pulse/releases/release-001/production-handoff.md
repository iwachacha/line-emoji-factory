- Item type: static-emoji

- Background: transparent.
- Export directory: releases/release-001/production/finals

## Visual Direction
- Round signal mascot with dark outline.
- Small face plus one status mark.
- Short ASCII label at the bottom.

## Slot Instructions
- OK: teal body, check mark.
- WAIT: yellow body, pause bars.
- DONE: green body, completion check.
- THX: coral body, small heart-like spark kept abstract.
- FYI: blue body, information dot.
- SOON: violet body, small clock arc.
- SOS: red body, attention triangle.
- LATER: gray body, return arrow.

## Export Checks
- Use RGB/RGBA PNG.
- Keep each file below 1 MB.
- Ensure transparent background is present.
- Avoid identical silhouettes by changing mark position and expression.
