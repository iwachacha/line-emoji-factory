# Pocket Pulse release-001 Usage Validation

## Cases
| Context | Emoji use | Baseline | Judgment |
| --- | --- | --- | --- |
| A teammate sends a task update. | OK | Plain "ok" is colder. | win |
| Someone asks for a quick pause. | WAIT | A long sentence interrupts. | win |
| A task is closed. | DONE | Text alone is enough but less visible. | win |
| A small favor is completed. | THX | Existing thanks stickers are often too emotional. | win |
| Sharing a light note. | FYI | Plain text can look abrupt. | win |
| Saying it will happen shortly. | SOON | Watch for promotional reading. | watch |
| Asking for help. | SOS | Needs a gentle, non-panic expression. | watch |
| Deferring a reply. | LATER | More polite than silence. | win |

## Judgment
- Structure: Pass.
- Brand: Pass.
- Product: Pass with watch on SOON and SOS wording.
