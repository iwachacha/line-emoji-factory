# Pocket Pulse release-002 Production Handoff

## Target
- Item type: static-sticker
- Content size: 370 x 320 PNG maximum, even dimensions.
- Main image: 240 x 240 PNG.
- Chat tab: 96 x 74 PNG.
- Background: transparent.
- Export directory: releases/release-002/production/finals

## Visual Direction
- Expand the release-001 emoji structure into a larger single-send sticker.
- Keep the same round signal mascot, dark outline, and short ASCII label.
- Make the face and status mark readable before the label.
- Do not make the design look like an app notification logo.

## Slot Instructions
- 01 OK: clear acceptance.
- 02 WAIT: calm pause.
- 03 DONE: complete and closed.
- 04 THX: light thanks.
- 05 FYI: gentle notice.
- 06 SOON: near-future timing.
- 07 SOS: help needed, not panic.
- 08 LATER: defer without rejection.

## Export Checks
- Width and height must be even.
- Content must not exceed 370 x 320.
- Main image must be exactly 240 x 240.
- Tab image must be exactly 96 x 74.
- Every PNG must stay below 1 MB.
