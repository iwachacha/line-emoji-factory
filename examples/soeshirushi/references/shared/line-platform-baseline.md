# LINE Platform Baseline

This file records the factory's current operational interpretation of LINE Creators Market emoji requirements. When official requirements change, update this file before changing validators or templates.

## Official Source Check

- Static emoji creation guideline checked 2026-04-26: https://creator.line.me/en/guideline/emoji/
- Static emoji set type and filename guideline checked 2026-04-26: https://creator.line.me/en/guideline/emoji/detail/
- Animated emoji creation guideline checked 2026-04-26: https://creator.line.me/en/guideline/animationemoji/
- Animated emoji frame and loop detail checked 2026-04-26: https://creator.line.me/en/guideline/animationemoji/detail/

## Static Emoji Baseline

- Tab image: 1 file, `96 x 74` px.
- Content images: `8`, `16`, `24`, `32`, or `40` files.
- Content image size: `180 x 180` px.
- Image format: PNG.
- Background: transparent.
- Resolution: at least 72 dpi.
- Color mode: RGB-compatible.
- One image file: `1MB` or less.
- ZIP upload: `20MB` or less.

## Animation Emoji Baseline

- Content images use APNG files with `.png` extension.
- Content APNG size: `180 x 180` px.
- One APNG file: `300KB` or less.
- Frame count: `5` to `20`.
- Total duration: `4` seconds or less.
- Loop count: `1` to `4`.
- The static tab image remains `96 x 74` px PNG.
- Static packaging remains the only supported package flow in this repo until animation release packaging is added.

## Metadata Baseline

- Creator name: 50 LINE-count characters or fewer.
- Emoji title: 40 LINE-count characters or fewer.
- Description: 160 LINE-count characters or fewer.
- Copyright: 50 LINE-count characters or fewer.
- Copyright uses ASCII letters and numbers only.
- Metadata must not include emoji characters.

## Factory Filename Interpretation

- Production filenames are internal and may be arbitrary.
- Submission filenames are generated by `tools/package-release.py`.
- Regular static emoji submission content images use zero-padded numeric names: `001.png`, `002.png`, ... up to the active set count.
- The filename numbering contract differs for kana / letter emoji package types; this factory currently supports regular emoji packaging only.
- The tab image is generated as `tab.png`.
- The LINE upload ZIP must contain only image files.

## Review-Risk Interpretation

- Promotional terms such as campaign, free, giveaway, sale, and discount are warnings.
- Brand, structure, and product fit must still be judged in the relevant rules and workflows.
