# item finalization Prompts

## Core Finish Prompt
- `そえしるし` の rough を、LINE 絵文字として完成度の高い静止絵文字に仕上げる。優先順位は 用途の読め方 → シルエット差 → 中央サインの視認性 → 細部の美しさ。太いアウトライン、角丸、中央サイン1個、脇役性を維持する。主力枠が補助枠より先に読めることを守る。文字を覆う背景帯や囲み枠にはしない。`

## Handoff-Specific Prompts
- `release-001 ことば整理コア16。16個すべてを同じ線の太さと温度感で整える。みて左 / みて右 は単体でも成立させ、補足 / 結論 と 確認 / 要返信 / 了解 は外形と中央サインを明確に分ける。重要 / 注意 / 締切 は色だけでなく外形差で判別できるようにする。主力14枠が補助2枠より前に読める強弱を保つ。`

## Fix Prompts
- `visibility fix: 中央サインが潰れる絵文字だけ、外形を保ったままサインを拡大し、線の太さを維持する`
- `difference fix: 似ている2絵文字は色を変えるのではなく、外形の切れ方と中央サインの形で差を立てる`
- `pair fix: 左右ペアが片割れに見える場合は、単体で完結する外形の閉じを追加する`
