# GPT / image_gen Rough Prompts

## Core Rough Prompt
- `そえしるし` の LINE 絵文字 rough board を作る。文字の主役性を崩さず、脇役として短文の意味を補助する記号ブランド。16個の静止絵文字セットを一覧で見せる。商品コンセプトは「重要度・依頼・状態・締めを、文字の脇で軽く整理するコア実用品セット」。角丸フック、タブ、旗、封緘マークを基礎にし、各絵文字は太い濃色アウトラインと中央の機能サイン1個で構成する。小表示でも差が読めるよう、外形差を大きく、装飾は最小限。組み合わせ補助枠は `みて左 / みて右` の1組のみ。`

## Variant Rough Prompts
- `release-001 ことば整理コア16 の rough board。含める機能は 重要、確認、お願い、共有、みて左、みて右、注意、補足、要返信、了解、決定、あとで、締切、メモ、おすすめ、結論。上段4個を強調し、主力14枠と補助2枠を見分けられるように並べる。白黒シルエット比較も入れる。文字を囲む背景帯や下線帯は作らない。各絵文字は単体送信でも意味が読めるようにする。`

## Negative / Guardrails
- `キャラクター化しない`
- `UIアイコンや企業ロゴに寄せない`
- `色違いだけで差を作らない`
- `左右ペアを片割れっぽくしすぎない`
- `左右ペア以外の組み合わせ補助枠を増やさない`
- `下線、背景帯、任意文字列を入れる枠を描かない`
