# release-001 rough board notes

## 目的
- `そえしるし` の外形文法と、`ことば整理コア16` の機能差分を1枚で確認する。

## board に必ず入れるもの
- 16個の一覧配置
- 上段4個の拡大比較
- `みて左 / みて右` の単体表示
- `みて左 + テキスト + みて右` の短語サンプル
- `重要 / 注意 / 締切` の白黒シルエット比較
- `補足 / 結論` の並置比較
- `確認 / 要返信 / 了解` の並置比較
- 主力14枠と補助2枠の見え方比較

## keep
- 角丸フック系の外形
- 太い濃色アウトライン
- 中央サイン1個
- 脇役として静かな温度感

## trim
- 飾り線
- グラデーション
- 外形の細かな凸凹
- 色でしか分からない差分

## drift
- キャラ化
- 囲み枠の完全化
- 片割れ感の強い左右ペア
- 補助枠の増やしすぎ
- UIアイコンやロゴへの接近
