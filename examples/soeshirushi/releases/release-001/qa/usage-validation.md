## 4. Repeating Failure Patterns
- 近い絵文字同士で混ざる箇所: `確認 / 要返信 / 了解`, `補足 / 結論`, `共有 / メモ`
- 句読点や既存記号で十分になってしまう箇所: `共有`, `メモ`, `左右ペア`
- スタンプや他商品に負ける箇所: `おすすめ` は感情温度が足りないと埋もれる
- 仕事寄り / 日常寄りの偏り: コアは仕事寄りに強い。日常側は `お願い / おすすめ / あとで` を丁寧に残す必要がある

- 強く直す絵文字: `S05`, `S06`, `S08`, `S09`, `S10`
- 切る候補: 現時点ではなし。`S15` は次の rough で再判定
- quality-ledger へ送る論点: 左右ペア単体成立、近機能3連続の見分け、`共有 / メモ / おすすめ` の勝ち筋
