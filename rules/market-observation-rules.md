# 市場観測証跡ルール

このファイルは、市場調査を一時的な印象で終わらせず、ブランド判断に使える証跡として残すための正本である。

## 観測に必ず残すもの
- 観測日
- 観測者
- store / platform
- 検索語
- locale
- category
- 混雑している領域
- 空いている領域
- 代表的な visual motif
- よくある title / keyword
- 模倣してはいけない境界
- 自ブランドが取れる差別化軸
- 初回 release 仮説

## 禁止
- 人気そう、売れそう、かわいい、だけで採用する。
- 競合の名前、構図、キャラクター特徴を近づける。
- 単発観測を恒久 rule に昇格させる。
- 市場観測をブランド核の代わりにする。

## factory 昇格条件
- 複数ブランドで同じ混雑領域へ寄っている。
- 同じ模倣リスクが繰り返し見える。
- market template の項目不足が原因で判断が曖昧になる。
