# 固定IP統制ルール

このファイルは、許諾済みの固定IPをLINE絵文字 / スタンプとして扱うときの判断基準の正本である。
権利侵害回避だけでなく、許諾範囲内で品質を守る制作統制を扱う。

## 必須確認
- 使用許諾範囲
- 使用可能キャラクター
- 使用可能ロゴ
- 使用可能モチーフ
- 禁止ポーズ
- 禁止表情
- 禁止文脈
- 色、線、顔比率、衣装、小物の固定仕様
- 公式素材との距離
- AI生成時の参照画像ルール
- brand canon へ転記する壊してはいけない要素
- シリーズごとに許容される変化
- 権利者レビューの要否
- 承認ログと差し戻し履歴

## Hard NG
- 許諾範囲にないキャラクター、ロゴ、素材を使う。
- 権利者が禁止した文脈や表情へ展開する。
- 公式素材を無断でトレース、加工、再配布する。
- AI生成の参照ルールが不明なまま制作する。
- 承認が必要な案件で承認ログがない。

## Revise
- キャラクター崩れが見える。
- 色、線、顔比率、衣装、小物の固定仕様が揺れている。
- LINE item としての即読性より、IP説明を優先している。
- 固定IPらしさはあるが、日常会話で使う場面が薄い。
